# Zay - Extended Frozen Ore

## Description
This mod extends the **Frozen Ore** smelting recipe in ICARUS to include additional ore types as possible random outputs.

Frozen Ore can be smelted at any furnace-type workstation and randomly yields one of the available ores. This mod adds **Platinum Ore** and **Titanium Ore** to that pool if they are not already present.

## Outputs Added by This Mod
- **Platinum Ore**
- **Titanium Ore**

## Recipe
Smelt **1x Frozen Ore** at any furnace to randomly receive **5x** of one ore from the full output pool.
The two outputs added by this mod:
- **1x Frozen Ore** -> 5x Platinum Ore
- **1x Frozen Ore** -> 5x Titanium Ore

> The output is selected randomly -- each smelt picks one ore type from the pool.

## Compatibility
- Version: 1.0.223
- Compatible with ICARUS week 223+ and future updates
