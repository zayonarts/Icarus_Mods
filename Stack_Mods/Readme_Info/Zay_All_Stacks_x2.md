# Zay - All Stacks [x2]

## Description
This mod multiplies the maximum stack size of every stackable item in ICARUS by 2.

Each item keeps its own unique stack limit — this mod simply scales all of them up by x2, rounded to a whole number.

## Compatibility
- Version: 1.4.223
- Compatible with ICARUS week 223+ and future updates
