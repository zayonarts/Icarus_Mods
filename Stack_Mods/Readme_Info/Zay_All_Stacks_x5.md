# Zay - All Stacks [x5]

## Description
This mod multiplies the maximum stack size of every stackable item in ICARUS by 5.

Each item keeps its own unique stack limit — this mod simply scales all of them up by x5, rounded to a whole number.

## Compatibility
- Version: 1.4.223
- Compatible with ICARUS week 223+ and future updates
