# Zay - All Stacks [50k]

## Description
This mod modifies all stackable items in ICARUS to have a maximum stack size of 50.000 units.

Instead of carrying just a few items at a time, you can now stack up to 50.000 of each item type in a single inventory slot. This significantly reduces inventory management and allows for more efficient resource gathering and storage.

## Compatibility
- Version: 1.4.222
- Compatible with ICARUS week 222+ and future updates
