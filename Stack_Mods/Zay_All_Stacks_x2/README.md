## 📋 Mod Details

| Feature | Detail |
|---------|--------|
| **Items Modified** | 1262 stackable items |
| **Stack Multiplier** | x2 from original values |
| **Scaling** | Per-item — each item keeps its unique ratio |
| **Data Table** | `D_Itemable` |
| **Mod Format** | `.EXMODZ` |
| **Multiplayer** | ✅ Single player, hosted MP & dedicated server |

> Each item's original `MaxStack` value is multiplied by **2** and rounded to a
> whole number. Items that originally stack to 100 will stack to 200, items
> that stack to 50 will stack to 100, and so on.

## 📥 Installation

1. Download `Zay_All_Stacks_x2.EXMODZ`
2. Import (Zay_All_Stacks_x2.EXMODZ) into your Icarus Mod Manager.

> **Note:** This mod uses the `.EXMODZ` format and requires Icarus Mod Manager to install.

## ⚠️ Disclaimer

- Backup and save all user data before installing or using any mods
- **Use at your own risk**
- If you filled extra slot capacities with this mod, loading without it may result in lost items
- Ensure you are running the latest version compatible with your Icarus version
