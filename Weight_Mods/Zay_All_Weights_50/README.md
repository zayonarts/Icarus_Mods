## 📋 Mod Details

| Feature | Detail |
|---------|--------|
| **Items Modified** | 3217 items |
| **Weight Reduction** | -50% from original values |
| **Rounding** | Rounded down to 1 decimal place |
| **Data Table** | `D_Itemable` |
| **Mod Format** | `.EXMODZ` |
| **Multiplayer** | ✅ Single player, hosted MP & dedicated server |

> Each item's original `Weight` value is reduced by **50%** and rounded down
> to one decimal place. For example, an item weighing 10.0 becomes 5,
> and an item weighing 1.0 becomes 0.5.

## 📥 Installation

1. Download `Zay_All_Weights_50.EXMODZ`
2. Import (Zay_All_Weights_50.EXMODZ) into your Icarus Mod Manager.

> **Note:** This mod uses the `.EXMODZ` format and requires Icarus Mod Manager to install.
