## 📋 Mod Details

| Feature | Detail |
|---------|--------|
| **Recipe Modified** | Frozen Ore (furnace smelting) |
| **New Outputs** | Platinum Ore, Titanium Ore |
| **Output Amount** | 5x per smelt |
| **Data Table** | `D_ProcessorRecipes` |
| **Mod Format** | `.EXMODZ` |
| **Multiplayer** | ✅ Single player, hosted MP & dedicated server |

## 🔬 How It Works

Smelting **1x Frozen Ore** at any furnace-type workstation will randomly
yield one of the available ores from the expanded output pool:

**Outputs added by this mod:**
- **Platinum Ore**
- **Titanium Ore**

All vanilla ore outputs remain intact — this mod only adds new possibilities.

## 📥 Installation

1. Download `Zay_Extended_Frozen_Ore.EXMODZ`
2. Import (Zay_Extended_Frozen_Ore.EXMODZ) into your Icarus Mod Manager.

> **Note:** This mod uses the `.EXMODZ` format and requires Icarus Mod Manager to install.
